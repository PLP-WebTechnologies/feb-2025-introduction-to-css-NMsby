What is CSS?

CSS stands for Cascading Style Sheets. It is a style sheet language used for describing the presentation of a document written in HTML. CSS is designed to enable the separation of presentation and content, including layout, colors, and fonts.

CSS makes it possible to adapt the presentation to different types of devices, such as large screens, small screens, or printers. CSS is independent of HTML and can be used with any XML-based markup language.

Key Features of CSS

  • Control layout of multiple web pages from one file
  • Apply different styles to different media types
  • Create responsive designs that adapt to different screen sizes
  • Precise control over typography and spacing
  • Animations and transitions for interactive elements

CSS Selectors

CSS selectors are patterns used to select the elements you want to style. Here are the main types of selectors:

Element Selectors

Select all elements of a specific type, like paragraphs, headings, etc.

Class Selectors

Select elements with a specific class attribute. Classes begin with a period (.) in CSS.

ID Selectors

Select a single element with a specific ID attribute. IDs begin with a hash (#) in CSS.

Pseudo-class Selectors

Select elements based on a specific state, like hover, focus, etc.
